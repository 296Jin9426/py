## print(objects)

```python
print("Hello,world!",1,'a',1+2,'a'+'b','a'*2,'b'*3)
```

## print(objects,sep=' ')

```python
print('a','b',sep=' ')
print('a','b',sep=',')
```

## print(objects,sep,end='\n')

```python
print('ab',end='\n')
print('---')
print('ab',end='.')
print('---')
print('ab',end='\n...')
print('---')
```
